# HaxBall Client - By Luq

Este cliente está diseñado para ofrecer la mejor experiencia de juego, eliminando restricciones y sumando herramientas clave en un solo lugar.

## 🚀 Instalación

Seguí estos pasos para dejar todo listo:

1. **Descargá el archivo** desde el enlace oficial:
   [Descargar HaxBall Client]()

3. Una vez descargado, buscalo en tu carpeta de descargas.

4. Hacé **click derecho** sobre el archivo y seleccioná **"Extraer aquí"**.

5. Entrá a la carpeta que se generó y ejecutá el programa para empezar a jugar.

---

## 🔥 Características Principales

Este cliente viene con optimizaciones exclusivas para mejorar tu rendimiento y herramientas avanzadas:

* **Unlocked FPS:** Jugá sin el límite estándar de frames para una fluidez total y una respuesta mucho más rápida.
* **HaxBall All-in-one Tool:** Una suite potente de utilidades integradas:
* **Búsqueda Avanzada de Salas:** Filtrá salas por nombre y usá "+" para buscar varios términos a la vez (ej: *espacio+soccer*).
    * **Auto Join:** ¿Sala llena? El cliente se encarga de reintentar y entrar apenas se libere un lugar.
    * **Traductor de Chat:** Botón integrado para traducir mensajes de otros idiomas al instante.
    * **Macros de Texto:** Configurá atajos rápidos (ej: escribís `.ad` y se transforma automáticamente en `!admin`).
    * **Grabación Rápida:** Presioná la tecla **R** en cualquier momento para empezar o dejar de grabar un replay.
    * **Chat Transparente y Toggle:** Ganá visión de campo con el chat traslúcido o escondelo completamente con la tecla debajo del **ESC**.
    * **Cámaras Extra:** Nuevos niveles de zoom disponibles usando las teclas **5, 6, 7 y 8**.
    * **Mute Local:** Silenciá a jugadores molestos haciendo click derecho en su nombre.
    * **Admin Shortcuts:** Si sos admin, botones de Kick y Ban rápidos con un solo click.
    * **Menciones:** Hacé click derecho en un mensaje o nombre para mencionar (@) a alguien rápido.
